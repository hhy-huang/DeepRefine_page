DeepRefine-Skill

An agent skill to evolve the quality of LLM-Wiki (Graphify) at test time. Maintained by HKUST-KnowComp.

Star us on GitHub

📦 Installation

Follow these steps to set up the repository on your local machine:

# 1. Clone the repository
git clone https://github.com/HKUST-KnowComp/DeepRefine-Skill.git

# 2. Enter the directory
cd DeepRefine-Skill

# 3. Install dependencies
pip install -r requirements.txt

🚀 Quick Start

You can run DeepRefine-Skill in different modes depending on your workflow. Below is an example of running it in Agent Mode:

python main.py --mode agent \
  --query "Explain the process of knowledge graph refinement" \
  --model gpt-4o

💻 Commands Overview

Here are some of the primary flags you can use with our entry script:

Argument Type Description
--mode String Set execution mode: agent or cli.
--query String The input task or query to be processed.
--model String LLM engine to use (e.g., gpt-4o, claude-3).
